# My React Messenger App

## Description
This is a responsive messaging application built with **React**, **TypeScript**, **CSS**, **Node.js** and **MySQL**, designed to provide smooth and efficient communication between users. The project is currently **under active development**, with new features being added continuously.

The application includes:

- A **chat interface** for one-to-one conversations
- **Persistent data storage** using MySQL
- A lightweight **Node.js/Express backend** for handling API requests
- **User search** functionality to quickly find friends by name
- A **responsive interface** built with modern React practices

## Features

- Send and receive messages
- Search users by name
- Persistent storage of users,chats and messages with MySQL
- REST API communication between frontend and backend
- Clean, modern messaging interface
- Ongoing development with additional messaging features planned

---

## Technologies Used

- **React** (functional components, hooks)
- **TypeScript** (type safety and better scalability)
- **CSS / Flexbox / Grid** for layout and responsiveness
- **Node.js + Express** for the backend API
- **MySQL** for persistent data storage
